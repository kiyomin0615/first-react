# Project Overview
A simple posting project
- Front End: React.js
- Back End(API): Express.js
  - [API](https://github.com/kiyomin0615/first-react-backend)
- Back End(DB): MongoDB

# Getting Started
## Prerequisites
Node.js

## Installation and Setup
1. Clone the repository  
```git clone <repository-url>```

2. Navigate to the project directory  
```cd <project-directory>```

3. Install dependencies  
```npm install```

4. Start the local server for front end  
```npm run dev```

5. Start the local server for back end  
Please refer to [API](https://github.com/kiyomin0615/first-react-backend)

6. Access the website running on local server  
```http://localhost:5173```
